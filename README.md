# **Palmoro Hotel**

Palmoro Hotel is a modern and elegant hotel booking website built with React and Vite.  
The project focuses on delivering a premium user experience with smooth animations, responsive layouts, and clean modern design inspired by luxury hotel platforms.

## 🌐 Reference Design
Palmoro Hotel Inspiration / Reference:  
https://palmoro.webflow.io/

---

## ✨ Features

- Modern Luxury Hotel UI
- Fully Responsive Design
- Smooth Animations with Framer Motion
- Form Validation using Formik & Yup
- Fast Navigation with React Router
- Reusable Components
- Clean and Organized Code Structure

---

## 🛠️ Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS
- Framer Motion
- React Router DOM
- Lucide React
- React Icons
- Formik
- Yup

### Development Tools
- ESLint
- PostCSS

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/malik-tanveer/palmoro-hotel.git

Go to the project folder:

cd palmoro-hotel

Install dependencies:

npm install

Start development server:

npm run dev

Build for production:

npm run build

---

The goal of this project is to practice modern frontend development and create a visually appealing hotel booking experience using the React ecosystem.
